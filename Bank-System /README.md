# 🏦 Bank Management System (C++)

A simple and interactive console-based Bank Management System built using C++.
This project demonstrates file handling, data structures, and CRUD operations using text files as a lightweight database.

# ✨ Features

- ✅ Show all clients
- ✅ Add new clients
- ✅ Update client information
- ✅ Delete clients
- ✅ Find clients by account number
- ✅ Store data using text files
- ✅ Interactive console menu system

# 🛠️ Technologies Used
- 💻 C++
- 📂 File Handling (fstream)
- 🧱 Structures (struct)
- 📦 Vectors
- 🔤 String Manipulation
- 🖥️ Console-Based UI
- 📁 Project Structure

The application stores client records inside a Clients.txt file using a custom separator format:

AccountNumber#//#PinCode#//#Name#//#Phone#//#Balance

# Each client record contains:

- 🆔 Account Number
- 🔐 Pin Code
- 👤 Client Name
- 📞 Phone Number
- 💰 Account Balance
